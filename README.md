# MLIP-FineTuning-DRX-Cathodes
Repository for “Dual Control of Percolation and Site Energy Distribution for Optimizing Li+ Transport in Disordered Rocksalt Cathodes”. This codebase includes active fine-tund MLIP (SevenNet) and dataset for DRX materials.

The raw datasets, including the LMTO training sets and test set are openly available on Zenodo at https://zenodo.org/records/19179419.
